À propos des Courriels Temporaires

Notre service de courriel temporaire utilise l'API mail.tm pour créer des adresses e-mail jetables qui expirent automatiquement après 24 heures.

Caractéristiques Principales

  • Protection de la Vie Privée : Gardez votre e-mail personnel privé et évitez le spam
  • Extraction Automatique du Code : Les codes de vérification sont automatiquement détectés et affichés
  • Domaines Multiples : Changez entre différents domaines de messagerie
  • Sécurisé : Aucune information personnelle requise pour créer un compte
  • Mode Sombre : Travaillez confortablement la nuit avec le mode sombre

Mode d'Emploi

  1. Cliquez sur « Nouveau Courriel » pour générer une adresse e-mail temporaire
  2. Utilisez cette adresse pour les inscriptions ou vérifications
  3. Les codes de vérification apparaîtront automatiquement
  4. Cliquez sur l'e-mail pour le copier dans le presse-papiers
  5. Cliquez sur « Vider la Boîte » pour réinitialiser la boîte de réception

Remarque : Les e-mails sont automatiquement supprimés après 24 heures. Ce service est destiné à un usage légitime uniquement.
