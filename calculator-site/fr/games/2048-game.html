Comment jouer :

Objectif : Créer une tuile avec le numéro 2048.

À propos du jeu 2048

2048 est un jeu de puzzle pour un seul joueur créé par Gabriele Cirulli en 2014. L'objectif est de créer une tuile avec le numéro 2048.

Règles du jeu

  • Déplacez toutes les tuiles dans une direction à l'aide des touches fléchées ou en glissant
  • Les tuiles avec le même numéro fusionnent quand elles entrent en collision
  • Chaque mouvement ajoute une nouvelle tuile (2 ou 4) dans un espace vide
  • Le jeu se termine quand le plateau est plein et qu'aucun mouvement n'est possible

Conseils pour réussir

Suivez ces conseils pour obtenir des scores plus élevés :

  • Planifiez vos mouvements à l'avance pour garder le plateau organisé
  • Gardez les tuiles de grande valeur dans un coin
  • Concentrez-vous sur la création de chaînes de fusions
  • Utilisez la fonction d'annulation pour corriger les erreurs
  • Soignez votre patience et ne précipitez pas vos mouvements
